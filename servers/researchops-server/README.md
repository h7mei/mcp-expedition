# ResearchOps MCP Server

Placeholder package for the future remote ResearchOps MCP server.

## Planned scope

- Streamable HTTP transport
- Authentication and authorization
- Shared ResearchOps domain resources for projects, risks, decisions, and tasks
- Remote tool surfaces complementary to the local Workspace server

## Current status

This package builds and exports a typed domain model only. It does **not** yet implement MCP transport, HTTP endpoints, or authentication.
