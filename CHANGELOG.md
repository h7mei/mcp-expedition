# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Conventional Commits](https://www.conventionalcommits.org/).

## [0.1.0] - 2026-07-13

### Added

- Monorepo foundation with pnpm workspaces and Turborepo
- Local Workspace MCP server over stdio
- Expedition CLI with `workspace inspect`, `server list-capabilities`, and `doctor`
- Shared schemas, configuration, permission engine, protocol tracer, and MCP client packages
- Sample workspace fixtures and initial documentation
